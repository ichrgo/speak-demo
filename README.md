# Speak Demo

## Setup
  * Install [Angular CLI](https://github.com/angular/angular-cli) on your local machine.
  * Clone repo and `cd` into directory.
  * Run `ng serve -o` from the command line.  Navigate to `http://localhost:4200/`.
